# Version Management Functional Area (Master Data Manager)
  In the Version Management functional area, administrators can validate data, review and reverse transactions, create versioned copies of data, and flag versions of data for subscribing systems. For more information about administrators, see [Administrators &#40;Master Data Services&#41;](administrators-master-data-services.md).
