## Commands
|     |     |
| --- | --- |
[azdata bdc spark batch](reference-azdata-bdc-spark-batch.md) | Manages batch operations within the Spark system.

[azdata bdc spark statement](reference-azdata-bdc-spark-statement.md) | Manages Spark statements.
