# Properties Property (ServerNetworkProtocolIPAddress Class)
  Gets the properties associated with an IP address.  
  
## Syntax  
  
```  
  
object  
.Properties [= value]  
```  
  
## Parts  
 *object*  
 A [ServerNetworkProtocolIPAdress Class](servernetworkprotocolipaddress-class.md) object that represents an IP address for the network protocol on an instance of [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)].  
  
## Property Value/Return Value  
 An array of objects that represent the properties supported by the IP address.
