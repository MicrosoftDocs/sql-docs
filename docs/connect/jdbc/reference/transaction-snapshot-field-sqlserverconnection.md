[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Used to specify the snapshot transaction isolation level.  
  
## Syntax  
  
```  
  
public static final int TRANSACTION_SNAPSHOT  
```  
  
## Field Value
